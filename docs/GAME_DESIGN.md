> **Версія:** 1.1.3  
> **Дата оновлення:** 2025-12-19

## 1. Концепція Гри

### 1.1 Elevator Pitch
> Переміщуй кольорові LEGO-подібні блоки та виводь їх через відповідні двері. Проста механіка, але складні рівні!

### 1.2 Жанр
- **Основний:** Puzzle / Головоломка
- **Піджанр:** Sliding Block Puzzle
- **Стиль:** Casual / Relaxing

### 1.3 Цільова Аудиторія
| Характеристика | Значення |
|----------------|----------|
| Вік | 6+ (всі віки) |
| Пол | Універсальний |
| Досвід | Casual gamers |
| Сесії | 1-5 хвилин |

### 1.4 Референси
- Rush Hour (класична настільна гра)
- Unblock Me (мобільна гра)
- Color Block Jam (оригінал)

## 2. Core Gameplay

### 2.1 Основна Механіка
```
┌─────────────────────────────────┐
│    [🔵 door]                    │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  │    🟡🟡🟡    🔵🔵       │    │
│  │    🟡        🔵         │    │
│  │                         │    │
│  │       🔴🔴🔴            │    │
│  └─────────────────────────┘    │
│              [🟡 door]          │
└─────────────────────────────────┘

Гравець drag-and-drop переміщує блоки.
Мета: вивести всі блоки через двері свого кольору.
```

### 2.2 Правила
1. **Переміщення:** Блоки рухаються по горизонталі та вертикалі
2. **Колізії:** Блоки не можуть проходити один через одного
3. **Вихід:** Блок виходить тільки через двері свого кольору
4. **Перемога:** Всі блоки виведені з поля

### 2.3 Game Loop
```
┌──────────────────┐
│   Вибір рівня    │
└────────┬─────────┘
         ▼
┌──────────────────┐
│  Завантаження    │◀──────────┐
└────────┬─────────┘           │
         ▼                     │
┌──────────────────┐           │
│   Ігровий цикл   │           │
│  - Drag блоку    │           │
│  - Перевірка     │           │
│  - Вихід блоку   │           │
└────────┬─────────┘           │
         ▼                     │
    ┌────┴────┐                │
    │ Перемога?│               │
    └────┬────┘                │
    Так  │   Ні                │
    ▼    └─────────────────────┘
┌──────────────────┐
│   Win Dialog     │
│  - Confetti      │
│  - Next Level    │
└──────────────────┘
```

## 3. Візуальний Дизайн

### 3.1 Арт-стиль
- **Стиль:** Flat design з LEGO-елементами
- **Настрій:** Яскравий, веселий, дитячий
- **Текстури:** Мінімальні, чисті кольори

### 3.2 Колірна Палітра
| ID | Назва | HEX | Використання |
|----|-------|-----|--------------|
| 0 | Red | #FF4444 | Блоки/Двері |
| 1 | Green | #44FF44 | Блоки/Двері |
| 2 | Blue | #4444FF | Блоки/Двері |
| 3 | Yellow | #FFFF44 | Блоки/Двері |
| 4 | Orange | #FF8844 | Блоки/Двері |
| 5 | Purple | #FF44FF | Блоки/Двері |
| 6 | Cyan | #44FFFF | Блоки/Двері |
| 7 | Pink | #FF88AA | Блоки/Двері |
| 8 | Lime | #88FF44 | Блоки/Двері |
| 9 | Brown | #AA6644 | Блоки/Двері |
| - | Frame | #8B4513 | Рамка поля |
| - | Background | #3d3d3d | Поле |
| - | Gradient | #667eea → #764ba2 | UI фон |

### 3.3 UI Елементи
| Елемент | Опис |
|---------|------|
| Блок | Суцільний колір + темна обводка + LEGO-кнопка в центрі |
| Двері | Кольорова смужка на рамці + стрілка напрямку |
| Рамка | Коричневе "дерево" з 3D ефектом |
| Кнопки | Rounded rectangles з gradient |

### 3.4 Анімації
| Анімація | Тривалість | Easing |
|----------|------------|--------|
| Block pickup | - | instant |
| Block move | - | instant |
| Block exit | 200ms | linear |
| Win dialog | 500ms | elasticOut |
| Confetti | 2000ms | linear |
| Star appear | 400ms | elasticOut |

## 4. Аудіо Дизайн

### 4.1 Звукові Ефекти (SFX)
| Звук | Тригер | Пріоритет |
|------|--------|-----------|
| Pickup | Взяття блоку | High |
| Drop | Опускання блоку | High |
| Slide | Рух блоку | Low |
| Exit | Вихід через двері | High |
| Win | Перемога | High |
| Button tap | Натискання кнопки | Medium |

### 4.2 Haptic Feedback
| Вібрація | Тригер | Інтенсивність |
|----------|--------|---------------|
| Light | Кнопки, вибір | Легка |
| Medium | Pickup блоку | Середня |
| Heavy | Drop блоку | Сильна |
| Success | Перемога | Патерн |

### 4.3 Музика
- **MVP:** Без фонової музики
- **Майбутнє:** Relaxing ambient loops

## 5. Прогресія

### 5.1 Структура Рівнів
| Діапазон | Складність | Особливості | Статус |
|----------|------------|-------------|--------|
| 1-5 | Tutorial | 2-3 блоки, прості форми | ✅ Верифіковано |
| 6-10 | Easy | 3-4 блоки, L-форми | ✅ Верифіковано |
| 11-15 | Medium | 4-5 блоків, складні форми | ✅ Верифіковано |
| 16-18 | Hard | 5+ блоків, ShortT/ShortL форми | ✅ Верифіковано |
| 19-27 | Hard | Hidden cells, складні комбінації | 🔄 Експортовано |
| 28-50 | Expert | Не експортовано | 📅 Заплановано |

### 5.2 Unlock System
```
Level 1 ─── unlocked by default
    │
    ▼ (complete)
Level 2 ─── unlocked
    │
    ▼ (complete)
Level 3 ─── unlocked
    │
    ...
```

### 5.3 Scoring (Майбутнє)
| Критерій | Зірки |
|----------|-------|
| Завершення | ⭐ |
| < N moves | ⭐⭐ |
| < N seconds | ⭐⭐⭐ |
